# My Portfolio — Starter

This repository contains a starter Next.js App Router portfolio scaffold built with TypeScript, Tailwind CSS, and Framer Motion. It is designed as a premium, editorial personal site and includes core pages, SEO-ready metadata, sitemap/robots routes, and MDX-ready blog support.

Quick start

```bash
npm install
npm run dev
```

Deployment

1. Push to a Git provider.
2. Connect the repo to Vercel and deploy (Vercel auto-detects Next.js projects).

What's included

- Next.js app router structure
- Tailwind CSS setup
- Framer Motion for tasteful motion
- Placeholder pages: Home, About, Services, Work, Blog, Teaching, Contact, Privacy, Terms, 404
- Sitemap and robots routes
